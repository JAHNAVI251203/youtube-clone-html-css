# YouTube Clone (HTML + CSS)

A static clone of the YouTube homepage built using **pure HTML and CSS**.
This project recreates the layout, styling, and responsive behavior of YouTube's main feed interface.

## Features

* YouTube-like homepage layout
* Responsive video grid
* Sidebar navigation menu
* Header with search bar and icons
* Video cards with thumbnails, titles, and channel info
* CSS Grid and Flexbox layout implementation

## Tech Stack

* HTML5
* CSS3
* Flexbox
* CSS Grid

## How to Run

1. Clone the repository

```
git clone https://github.com/JAHNAVI251203/youtube-clone-html-css.git
```

2. Open the project folder

3. Open **index.html** in your browser
   or run with **Live Server** in VS Code.
